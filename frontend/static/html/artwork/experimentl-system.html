Custom fractal writer

As my first bespoke piece, I have given the user the opportunity to create their own fractal patterns using a rule builder.
This system is able to iterate 1 to 8 times using two rule-sets that the user has control over.
The user then will input combinations of A and B in the string, choose line length and then press draw!
The alphabet for this system goes as follows:

  • F: This will draw a line going in the direction assigned.
  • +: This will turn the pointer clockwise by the number of degrees assigned.
  • -: This will turn the pointer anticlockwise by the direction assigned
  • A: This will make a recursive call to Rule A
  • B: This will make a recursive call to Rule B
  • [: This will save the current point, this is useful for creating branched fractals
  • ]: This will restore the point back to where it was saved after calling [

See below for examples:

One [B]+F++F-[B]+B+++ F++F-F+F A ABABABA 60 30 4 20 0
Two F++F-F[B]--F++ F++F-F+F A ABBABAAB 60 30 4 20 0
Three F[BFF+]F-- FF--F+FF A ABBBA 60 30 5 10 0

Press draw a few times and see what kinds of shapes and trails you can come up with!

RULES:

  • No blank fields, if you wish to not rotate, do not call + or - in your rules
  • Rule B cannot call Rule A if Rule A is calling Rule B.
  • The program will not draw if you haven't clicked on the canvas and set a point.
