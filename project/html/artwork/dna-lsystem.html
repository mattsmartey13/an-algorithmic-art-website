L-System DNA

We are now able using modern science to 3d print DNA, a new technique which has been used to develop the COVID-19 vaccine. The question beggars: what combination of DNA nucleotide bases equates to what? Once you open with many A's and C's for instance, will that head towards a specific type of protein? Or is there some kind of production rule? This inspired me further to see if I could generate my own DNA double helix using L-system production rules.

This does not contain an axiom and production rule, but a mechanism that will allow bases to be called on following each base insertion. Using the ATGC language, you can specify which bases are to be called out following the insertion of a base. The iteration mechanism will allow you to generate a hefty string. This will cause the backbone and nucleotide bases to shrink, and the DNA strand will continue below. Once the strand hits the end of the page, it will go down one level and then continue in the reverse direction:

  • Right -> Left
  • Left <- Right

And so forth. This piece was fascinating to create because I devised its development process like the making of a protein. Proteins have primary, secondary, tertiary and quaternary structures:

Primary
The chain of amino acids bonded to one another
Secondary
Individual chains of amino acids bonded by hydrogen bonds between one another
Tertiary
A large amalgamation of hydrogen and ionic bonds will give the protein its 3D structure
Quaternary
Multiple tertiary structure proteins joining one another to form a larger unit

I have used this as a methodology to develop the DNA by breaking it into stages.

Primary
Create methods to draw the backbone and nucleotide bonds using coloured lines and bezier curves
Secondary
Use this to draw a single helix with a number of nucleotides
Tertiary
With enough iterations, can we fill the canvas width with a strand of DNA?
Quaternary
Create a method to bounce the helix on a lower level to give the appearance of bonding between strands.

In terms of ATGC, which bases are called following a base being added into the helix? Start with an axiom using a single nucleotide base, the iteration system will concatenate a single DNA strand based on the iteration system. A second strand will then be constructed based on the matching nucleotide. For best results, input 2-4 bases.
