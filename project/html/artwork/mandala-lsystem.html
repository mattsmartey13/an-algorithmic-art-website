L-System Mandala

Using the L system language of this project, you can now construct a Mandala. The mandala piece in the preface section utilised a different and more simple algorithm: the program would simply plot the range of specified points on the canvas using a set of trigonometry rules, and then draw a line from each point to the other points. This piece however will utilise L systems that originate from the centre of the canvas and traverse in all 360 degrees, before being called recursively at the finishing point of each drawn L-system.
