Garden of L-systems

For my second art piece, I have used my research into L-system and inspiration from Aristid Lindenmayer's "The Algorithmic Beauty of Plants" to allow the user to create a garden from L-system algorithms. The user has no input on the L-system language, so each garden piece is randomly generated. The user however has the choice of placement, so following the selection of a type (cloud, tree etc), once the user has the canvas set up and clicks on the canvas, the plant life will be generated there. Each tree, flower and blade of grass will be different yet exhibit self-similarity.
