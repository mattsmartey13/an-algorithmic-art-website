Custom fractal writer

As my first bespoke piece, I have given the user the opportunity to create their own fractal patterns using a rule builder. The user is prompted to enter an axiom and a production rule. Although very similar, there are two alphabets to consider:

Axiom alphabet

  • F : This will be substituted for the production string.
  • + : This will turn the pointer clockwise by the number of degrees assigned.
  • - : This will turn the pointer anticlockwise by the direction assigned.

Production alphabet

  • F : This will draw a line going in the direction assigned.
  • + : This will turn the pointer clockwise by the number of degrees assigned.
  • - : This will turn the pointer anticlockwise by the direction assigned.

For example, with an axiom of F-F+F and a production rule of FF-F+F-F+F, every F in the axiom will be substituted for the production rule.

Examples

Algorithm name Axiom Production Rotational angle
Pentadendrite F-F-F-F-F F-F-F++F+F-F 72
Icy fractal F+F+F+F FF+F++F+F 90
Koch curve F F+F--F+F 80

RULES:

  • Spaces in the axiom and production strings are not compiled
  • Only the F, + and - symbols are interpreted
  • If the canvas isn't clicked, it will default to the top corner
